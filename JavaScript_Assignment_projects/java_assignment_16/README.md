# Product Information Web App

## Overview
This web application displays product information stored in an XML file and allows users to search for products by name.

## Files
- `products.xml` - Contains the product data.
- `product.html` - The main HTML file that contains the structure of the page.
- `product.js` - JavaScript file that handles the functionality (parsing XML, displaying data, searching).

## Usage
1. Open `product.html` in a web browser.
2. The table will display all the products.
3. Use the search bar to filter products by name.
